# File Integrity Monitor
Intern ID: CITS5240

This is a cybersecurity mini-project developed using Python.

## Features

* Creates baseline hashes for files
* Detects modified files
* Detects newly added files
* Detects deleted files
* Uses SHA-256 hashing
* Monitors file changes in a selected folder

## Technologies Used

* Python

## How to Run

```bash
python file_integrity_monitor.py
```
